User Whitelist

The user whitelist specifies which domains you trust not to track your browsing.

This list assumes that domains will be entered without protocol, and with only commas separating the domains.

Examples:

  • priv.ly
  • priv.ly,privly.org
  • subdomain.privly.org,priv.ly,privly.org



Content Servers

Set which domain you upload your content to when generating new links directly from the extension.




These are the recommended servers during development, but if you select "other" you will be able to assign it to be any domain you like. Please be certain that the custom domain is trustworthy and keeping up with system development. Learn More.
